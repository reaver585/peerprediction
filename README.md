# PeerPrediction

Python module for evaluation of subjective truthfulness of peer prediction mechanisms.

Dependencies for the module are:

* python >= 3.6

* numpy >= 1.13.3

* matplotlib >= 2.0.2

* pyjags >= 1.2.2

Additional dependencies for pyjags might include:

* JAGS >= 4.3.0

* gcc >= 4.8.1
